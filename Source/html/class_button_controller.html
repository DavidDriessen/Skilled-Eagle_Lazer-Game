Button Controller.

Button Controller contains 2 states. It starts in the not shooting state. If the button is pressed button is high and state is not shooting so it calls the game controller function buttonPressed() and state changes to shooting. If button is released and state was shooting the buttonReleased() function of the game controller is called and state changes to not shooting.
